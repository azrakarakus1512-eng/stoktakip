# 📖 Projenin Amacı ve Kısa Açıklaması

Stitch Stok Takip Uygulaması, işletmelerin ürünlerini kolayca yönetebilmesi amacıyla geliştirilmiş modern, mobil uyumlu ve kullanıcı dostu bir stok yönetim sistemidir.

Uygulama sayesinde kullanıcılar;

- Yeni ürün ekleyebilir.
- Ürün bilgilerini güncelleyebilir.
- Ürün silebilir.
- Stok giriş ve çıkış işlemleri yapabilir.
- İşlem geçmişini görüntüleyebilir.
- Marka bazlı filtreleme gerçekleştirebilir.
- Kritik stok seviyelerini takip edebilir.
- Ürün görselleri yükleyebilir.

Proje hem **LocalStorage** kullanan tarayıcı tabanlı sürüm hem de **Node.js + SQLite** kullanan sunucu tabanlı sürüm olarak geliştirilmiştir.

# ⚙ Kurulum Adımları

1. Projeyi GitHub üzerinden bilgisayarınıza klonlayın.

```bash
git clone https://github.com/azrakarakus1512-eng/stoktakip.git
```

2. Proje klasörüne girin.

```bash
cd stoktakip
```

3. Gerekli paketleri yükleyin.

```bash
npm install
```

4. Sunucuyu başlatın.

```bash
node server.js
```

5. Tarayıcıdan aşağıdaki adrese gidin.

```
https://stitchstoktakip.vercel.app/
```
